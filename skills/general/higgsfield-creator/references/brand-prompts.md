## SECTION 1: AS MOODBOARD RECIPE

### What the Moodboard Does
The Moodboard trains Higgsfield's AI on your visual style, palette, and atmosphere. For Aviation Synergy, the moodboard encodes three things: the deep-navy-to-teal color signature, the precision-cinematic aesthetic (clean, high-contrast, no visual clutter), and the APAC aviation authority environment.

**A well-built AS Moodboard eliminates the need to describe color grade and tone in every prompt.** Simply select the moodboard before generating and the AI inherits the full visual language.

### Reference Image Categories to Curate (Target: 25–30 images total)

**Do NOT include human faces in the moodboard.** Faces belong in Soul Cast or Soul ID. Moodboard references should be environments, textures, color palettes, and lighting studies.

#### Category 1: Aviation Intelligence Environments (8–10 images)
- Modern control rooms with multi-screen setups (blue/teal dominant lighting)
- Aviation operations centers, flight operations rooms, NOC (Network Operations Center) spaces
- Dark-themed data center environments with glowing screens
- Airport control towers at night — interior views
- Flight deck instruments close-up (teal/amber instrument lighting)

**Where to source:**
- Search "aviation control room night blue teal" on Unsplash, Pexels, or premium stock
- Search "flight operations center night screens"
- Search "dark NOC network operations center blue lighting"

#### Category 2: Cinematic Color References — Deep Navy + Teal (8–10 images)
- Film stills or cinematography reference images with strong navy/teal palette
- Architecture photography: glass buildings at night with teal reflections
- Underwater photography (natural deep blue-teal palette)
- Tech product photography on dark backgrounds with teal accent lighting

**Target color profile for these images:**
- Dominant: deep blue-black and dark navy in shadows
- Accent: bright teal/cyan as key light or screen glow
- Secondary: pale teal or light teal as fill/bounce
- Minimal warm tones (small touches of amber in practical lights are fine)

#### Category 3: Precision Cinematic Texture (5–8 images)
- Premium automotive photography (dark studio + precise lighting — similar compositional precision)
- Aerospace hardware close-ups (aircraft engines, instruments, precision components)
- Data visualization art / infographic imagery with dark backgrounds and glowing elements
- Architecture: airports, terminals, modern APAC commercial buildings at night

#### Category 4: APAC Aviation Context (4–5 images)
- Aerial shots of APAC cities at dusk (Singapore, Hong Kong, Bangkok, Tokyo)
- Airport tarmac wide shots at golden hour or blue hour
- Commercial aircraft on apron with dramatic sky

### Uploading the AS Moodboard
1. Navigate to → https://higgsfield.ai/moodboard/upload
2. Upload all 25–30 curated reference images as a batch
3. Name it: `Aviation-Synergy-Brand-v3.1`
4. Save and note the moodboard ID for project use

### Moodboard Maintenance
- Create a new moodboard version if brand guidelines update (v3.2, v4.0, etc.)
- Keep the reference images in a shared folder so the moodboard can be rebuilt if needed
- The moodboard strengthens with more coherent references — always prioritize style consistency over quantity

## SECTION 2: SOUL HEX COLOR SIGNATURE

### What Soul HEX Does
Soul HEX extracts precise color palettes from reference images and applies them to all subsequent generations. For Aviation Synergy, it locks the deep navy → sky teal → coral gradient signature without requiring color prompting.

### AS Soul HEX Reference Images (Upload 15–20 images)

The goal is to feed Soul HEX images where the AS brand palette appears naturally. The system extracts:
- The dominant shadow color (should be deep navy #1A2744)
- The primary accent/highlight color (should be sky teal #1A9BB5)
- The secondary accent (should be light teal #5EC8DC)
- The alert/emphasis color (should be coral #E8845A)

**Best reference types:**
- Screenshots of Aviation Synergy's own dashboards and UI (if available) — these directly encode the exact palette
- Teal-lit data visualization imagery (same palette as AS brand)
- Deep blue-to-teal gradient photography
- Any imagery where the AS brand palette appears naturally in practice

**Workflow:**
1. Navigate to → https://higgsfield.ai/image/soul-cinematic
2. Open the Color Transfer tab
3. Upload 15–20 AS palette reference images
4. System extracts and saves the color signature
5. Label it: `AS-Brand-v3.1-Color`
6. Apply at generation time by selecting this saved color signature

### Soul HEX vs. Moodboard — Know the Difference

| | Moodboard | Soul HEX |
|---|---|---|
| What it captures | Overall style, tone, atmosphere, composition tendency | Color palette and color relationships only |
| How many images | 20–30 | 15–20 |
| What it influences | Full visual language (lighting, mood, grain, composition) | Color grading and palette |
| Best used for | Complete style lock | Precise color consistency across clips |

**For maximum Aviation Synergy brand fidelity: use BOTH together.** Moodboard for the full visual language, Soul HEX for color precision.

## SECTION 5: BRAND DON'TS

Always include in prompt negatives for Aviation Synergy content:

```
no stock footage aesthetic, no startup visual clichés, no oversaturated HDR, no generic corporate visuals, no text overlays, no watermarks, no lens flare overuse, no plastic textures, no cartoon or illustrated style, no obvious AI artifacts, no morphing, no distortion
```

**Tone don'ts (for any AI-generated copy accompanying the content):**
- No "game-changing", "disruptive", "ninja", "revolutionary"
- No Silicon Valley startup energy
- Use instead: "predictive cycles", "systems architecture", "latent risk", "safety loop closure", "APAC aviation intelligence"
